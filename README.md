# 📊 Sales Prediction using Machine Learning

## 📌 Project Overview

This project focuses on predicting product sales using machine learning based on advertising budgets. It analyzes how spending on TV, Radio, and Newspaper advertisements influences sales and builds a regression model to estimate future sales.

---

## 🎯 Objective

The main objectives of this project are to:

- Analyze the relationship between advertising budgets and sales.
- Explore the dataset using data visualization techniques.
- Build a machine learning model for sales prediction.
- Evaluate the model using standard regression metrics.
- Understand how advertising impacts business growth.

---

## 📂 Dataset

The dataset contains advertising budgets and corresponding sales values with the following features:

- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget
- Sales
